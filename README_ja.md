# GBF Battle Analyzer

**日本語** | [English](./README.md)

**グランブルーファンタジー用 リアルタイムバトルログ解析ツール**

この拡張機能は、ブラウザ内でバトルデータをリアルタイムにキャプチャ・解析し、ダメージ分布やターン履歴などの洞察を提供します。

## 🚀 インストール

**ベータ版:** [Chrome Web Store](https://chromewebstore.google.com/detail/gbf-battle-analyzer/fbbnbplpljbbdompckjckhdmmkmodnhc) 

## ✨ 機能

- **リアルタイムログ解析**: 読みやすくフォーマットされたバトルログを即座に表示します。
- **ダメージ分布**: キャラクター、アビリティ、攻撃タイプごとのダメージ内訳を可視化します。
- **ターン履歴**: 過去のターンやアクションを振り返ることができます。
- **プライバシー重視**: すべてのデータはお使いのブラウザ内にローカル保存されます。外部サーバーへの送信は行われません。
